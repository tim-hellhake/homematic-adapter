Connect your HomeMatic devices.

## Configuration
Set the ip address or host name of your ccu in the settings and the port if it deviates from the standard port.

## Supported devices
- HM-CC-RT-DN

## Teach in device
- Go to the _Add things_
- A new thing should appear
- Add it
